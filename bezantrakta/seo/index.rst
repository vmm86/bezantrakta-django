Общая информация
================

.. only:: dev

  Динамически генерируемые "статические" файлы (иконки, *json*, *txt*, *xml*), необходимых для представления сайта в Интернете и SEO-оптимизации. Содержимое файлов генерируется динамически "на лету", по необходимости используя специфичную информацию конкретного сайта, открытого в данный момент.
