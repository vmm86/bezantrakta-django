Общая информация
================

Приложение содержит SVG-шаблон для создания электронных билетов, отправляемых на email покупателю при выборе соответствующего способа заказа билетов.

Генерация PDF-билетов
---------------------

Логика генерации билета, создания штрих-кодов и QR-кодов содержится в модуле ``shortcuts``.

.. automodule:: bezantrakta.eticket.shortcuts
