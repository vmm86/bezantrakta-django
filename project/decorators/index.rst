Декораторы
##########

Общие декораторы для всего проекта.

*********************
default_json_settings
*********************
Получение JSON-настроек по умолчанию при создании новой записи в модели. Декорирует пустую *callable*-функцию, которая указывается в параметре ``default`` для требуемого поля модели. Используется при создании новых экземпляров моделей, у которых есть поле с JSON-настройками (сайты, СПБ, СОО, события/группы) для удобного заполнения этого поля настройками по умолчанию, которые хранятся в модуле ``settings`` в корне соответствующего приложения.

.. automodule:: project.decorators.default_json_settings

***************
queryset_filter
***************
Фильтрация записей в какой-либо модели по специфическому полю *field_name* другой модели *model_name*, связанной с текущей моделью по соответствующему внешнему ключу. Применяется для фильтрации *queryset* в админ-панели по выбранному в выпадающем списке городу/сайту.

.. automodule:: project.decorators.queryset_filter
