Настройки проекта
=================
Настройки проекта находятся в пакете ``settings`` и разбиты на несколько модулей.

Собственно файл с настройками проекта в текущем окружении ``project.settings.settings`` является символьной ссылкой на 2 уровня выше. Это сделано для  удобного разворачивания новых версий проекта, чотбы файл с настройками фактически находится ВНЕ проекта и не затрагивается при очередном обновлении.

Шаблоны настроек ``development``, ``staging`` и ``production``-окружения находятся в отдельных файлах. При первоначальном запуске проекта один из 3-х файлов указанных шаблонов берётся за основу, переименовывается в ``settings`` и помещается на 2 уровня выше пакета ``project.settings``. Возможные последующие правки настроек делаются в файле настроек проекта и добавляются в одном из 3-х шаблонов настроек в системе контроля версий.

Секретный ключ ``SECRET_KEY``, без которого проект не запустится, при первом запуске генерируется случайным образом и записывается в модуль ``project.settings.simsim``, НЕ находящийся в системе контроля версий, а затем импортируется из него.

Кастомные параметры:

* ``BEZANTRAKTA_ADMIN_URL`` (str): Кастомный адрес для админ-панели вместо стандартного ``admin`` (``simsim``).
* ``BEZANTRAKTA_CATEGORY_ALL_SLUG`` (str): Псевдоним категории событий *"Все события"* (``vse``).
* ``BEZANTRAKTA_CATEGORY_ALL_TITLE`` (str): Название категории событий *"Все события"* (``Все события``).
* ``BEZANTRAKTA_ORDER_VIEWS`` (tuple): Виды, являющиеся шагами заказа билетов 1, 2 и 3. При их выполнении не должны работать процессоры контекста в приложении ``event`` для работы с событиями в базовом шаблоне (``event``, ``checkout``, ``confirmation``).
* ``BEZANTRAKTA_ETICKET_PATH`` (str): Путь для сохранения электронных билетов в формате PDF (``PARENT_DIR/e_tickets``).
* ``BEZANTRAKTA_ROOT_DOMAIN`` (str): Корневой домен для всех сайтов (``bezantrakta-dev.rterm.ru``).
* ``BEZANTRAKTA_ROOT_DOMAIN_SLUG`` (str): Псевдоним сайта, работающего на корневом домене (``vrn``).

Значения по умолчанию:

* Реквизиты организатора
    * ``BEZANTRAKTA_DEFAULT_PROMOTER_TITLE`` (str): Название.
    * ``BEZANTRAKTA_DEFAULT_PROMOTER_INN`` (str): ИНН.
    * ``BEZANTRAKTA_DEFAULT_PROMOTER_OGRN_IP`` (str): ОГРН или ОГРНИП.
* Реквизиты продавца
    * ``BEZANTRAKTA_DEFAULT_SELLER_TITLE`` (str): Название.
    * ``BEZANTRAKTA_DEFAULT_SELLER_INN`` (str): ИНН.
    * ``BEZANTRAKTA_DEFAULT_SELLER_OGRN_IP`` (str): ОГРН или ОГРНИП.
* ``BEZANTRAKTA_DEFAULT_MAX_SEATS_PER_ORDER`` (int): Максимальное число билетов в одном заказе (``7``).
* ``BEZANTRAKTA_DEFAULT_HEARTBEAT_TIMEOUT`` (int): Таймаут в секундах на повторение запроса для получения списка свободных мест и списка цен на билеты в событии (``10``).

.. todo:: Переименовать в ``BEZANTRAKTA_DEFAULT_SEATS_TIMEOUT``.

* ``BEZANTRAKTA_DEFAULT_SEAT_TIMEOUT`` (int): Таймаут в минутах на добавление места в предварительный резерв, по истечении которого место автоматически освобождается, если такой функционал предусмотрен в текущем сервисе продажи билетов (``15``).
