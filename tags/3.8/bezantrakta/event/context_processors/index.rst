####################
Процессоры контекста
####################

***************************
Афиши в больших контейнерах
***************************
Афиши событий/групп, размещённые в больших контейнерах, выводятся на любых страницах в основном шаблоне сайта, поэтому они постоянно находятся в выходном контексте с помощью процессора контекста ``big_containers``.

.. automodule:: bezantrakta.event.context_processors.big_containers

************************
Список категорий событий
************************
Список категорий выводится в основном шаблоне сайта и постоянно находится в выходном контексте с помощью процессора контекста ``categories``.

.. automodule:: bezantrakta.event.context_processors.categories
