##############################
Документация для разработчиков
##############################

Новая платформа для продажи билетов http://bezantrakta.ru/ создавалась для того, чтобы:

* преодолеть имеющие в предыдущей версии системы недостатки,
* предоставить новые возможности для продажи билетов на сайте.

Основа проекта - фреймворк **Django 1.11** (стабильная LTS-версия) и **Python 3.5** (без обратной поддержки Python 2.7) в отдельном виртуальном окружении. Для каждой новой мажорной версии проекта *желательно создавать заново* копию базы данных (например, ``belcanto_bezantrakta_django_3``) и виртуальное окружение (например, ``venv_v3``).

Описание первоначального развёртывания и дальнейшей разработки/поддержки содержится в файле :doc:`README`.

*****************
Структура проекта
*****************

* Настройки проекта и общие компоненты находятся в пакете ``project``.
* API сайта для работы с заказами билетов (на данный момент внутреннее, т.е. не обязательно работающее наружу) находится в приложении ``api``.
* Приложения, касающиеся внутренней логики работы сайтов Безантракта, находятся в пакете ``bezantrakta``.
* Приложения для работы со сторонними сервисами с использованием их *API* находятся в пакете ``third_party``.
