–°–∞–π—Ç—ã
-----
–°–∞–π—Ç –ø–æ –≤–Ω–µ—à–Ω–µ–º—É –∫–ª—é—á—É –ø—Ä–∏–≤—è–∑—ã–≤–∞–µ—Ç—Å—è –∫ –≥–æ—Ä–æ–¥—É, –≤ –∫–æ—Ç–æ—Ä–æ–º—É –æ–Ω –æ—Ç–Ω–æ—Å–∏—Ç—Å—è.

*–ò–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä* —Å–∞–π—Ç–∞ (int) - —á–∏—Å–ª–æ –∏–∑ 3-—Ö —Ü–∏—Ñ—Ä. –ü–µ—Ä–≤—ã–µ 2 —Ü–∏—Ñ—Ä—ã - *–∞–≤—Ç–æ–º–æ–±–∏–ª—å–Ω—ã–π –∫–æ–¥ —Ä–µ–≥–∏–æ–Ω–∞*, –≤ –∫–æ—Ç–æ—Ä–æ–º –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≥–æ—Ä–æ–¥ —Å–∞–π—Ç–∞. –¢—Ä–µ—Ç—å—è —Ü–∏—Ñ—Ä–∞ - –ø–æ—Ä—è–¥–∫–æ–≤—ã–π –Ω–æ–º–µ—Ä –æ—Ç ``0`` –¥–æ ``9``.

**üëâ** –ß—Ç–æ–±—ã —Å–æ–∑–¥–∞—Ç—å "–æ—Å–Ω–æ–≤–Ω–æ–π" —Å–∞–π—Ç –≤ –≤—ã–±—Ä–∞–Ω–Ω–æ–º –≥–æ—Ä–æ–¥–µ:
